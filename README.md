# Seabreeze Chatbot – Propello AI Internship Assignment

An AI-powered Real Estate lead chatbot built for **Seabreeze by Godrej Bayview**, Vashi.

## Built By
Vyshavani Reddy – AI/Tech Intern Applicant @ Propello AI

## Features
- Conversational AI assistant (Nova) powered by Claude API
- Answers project-specific questions (pricing, amenities, location)
- Natural lead capture – collects Name, Budget & Configuration
- Quick-reply chips for faster UX
- Responsive design – works on mobile & desktop

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Groq API (Llama 3.3 70B)


## How to Run
1. Clone this repo
2. Add your API key in `script.js` (line 1)
3. Open `index.html` with Live Server in VS Code
